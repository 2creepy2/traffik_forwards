Forward all traffik from one vps (proxy) to another using IPv4.
Run this scripts on vps, that you choose to use as proxy.
To install use command 'sudo ORIGIN_IP="xxx.xxx.xxx.xxx" bash -c "$(curl -sSL https://raw.githubusercontent.com/2creepy2/traffik_forwards/main/forwarding_install.sh)"' change origin ip to your destination vps ip.
To uninstall use command 'sudo bash -c "$(curl -sSL https://raw.githubusercontent.com/2creepy2/traffik_forwards/main/forwarding_delete.sh)"'
